# Movie Watchlist App

## Features
- Search movies using OMDB API
- Add/remove movies
- Mark as watched
- Rate movies
- Persistent storage with localStorage

## Tech Stack
React, JavaScript, CSS, REST API

## Setup

Create a `.env` file:

VITE_OMDB_API_KEY=your_api_key_here
